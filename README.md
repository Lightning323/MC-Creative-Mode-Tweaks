# MC-Creative Mode Tweaks
[![Custom Badge](https://img.shields.io/badge/Download-CurseForge-orange?style=for-the-badge&logo=curseforge)](https://www.curseforge.com/minecraft/mc-mods/creative-mode-tweaks)

Improves the Creative Mode experience by adding subtle enhancements and quality-of-life features. Every feature is designed to blend in with the vanilla experience and is highly configurable, allowing players to fine-tune settings to their preference via a config file.

## Current Features
-  Noclip Mode – Move seamlessly through blocks while in Creative Mode.
-  Extended Reach Distance – Interact with blocks and entities from much farther away.
-  Replace Tool – Instantly swap blocks without breaking and placing them manually.
-  Disabled Flight Inertia – Fly with precision, eliminating the momentum delay when stopping.
